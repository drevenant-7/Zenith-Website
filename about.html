📖 What is Zenith Launcher?

Zenith Launcher is a powerful Minecraft: Java Edition launcher for iOS and iPadOS devices. This is a personal fork maintained by @drevenant-7, based on @congcq's Pojav Patch, which itself is forked from PojavLauncher Team's iOS version.

Note: While Zenith allows internal account usage, we strongly encourage supporting Mojang by purchasing a legitimate Premium Account.

✨ Features

🎮 Gameplay

  • ✅ Supports Minecraft versions from early beta to latest snapshots
  • ✅ Forge, Fabric, OptiFine & Quilt mod loader support
  • ✅ Microsoft account & demo mode login
  • ✅ Customizable on-screen controls
  • ✅ Full keyboard, mouse & game controller support

⚙️ Technical

  • ✅ Optimized for jailbroken & TrollStore devices
  • ✅ Just-In-Time (JIT) compilation support
  • ✅ Increased memory limit capabilities
  • ✅ Hardware-accelerated rendering
  • ✅ Extensive mod compatibility
