🔧 Installation

Choose the installation method that works best for your device:

🎯 Method 1: TrollStore (Recommended)

Best Performance

TrollStore provides the best experience with automatic JIT and no re-signing required.

Requirements:

  • iOS 14.0 - 17.0 (for TrollStore 1)
  • iOS 15.0 - 17.0 (for TrollStore 2)
  • A computer for initial setup

Steps:

  1. Install TrollStore on your device following the official guide
  2. Download the latest Zenith Launcher IPA from GitHub Releases
  3. Open the IPA with TrollStore
  4. Launch Zenith Launcher - JIT will be enabled automatically!

📦 Method 2: AltStore / SideStore

Free

Free alternative that requires re-signing every 7 days.

Requirements:

  • Apple ID (free or developer)
  • Computer with AltServer installed

Steps:

  1. Install AltStore on your device
  2. Download the Zenith Launcher IPA
  3. Open in AltStore and install
  4. Refresh every 7 days to keep it working

⚡ Enabling JIT

Just-In-Time (JIT) compilation is required for optimal performance. Here's how to enable it:

Method External Device Wi-Fi Required Auto-Enabled
TrollStore ❌ No ❌ No ✅ Yes
JitStreamer ❌ No ✅ Yes ✅ Yes
SideJIT ✅ Yes ❌ No ❌ Manual
iOS 17.x - 18.x: A computer with Python 3.11+ and pymobiledevice3 is required. See JIT Documentation for details.

🔑 Signing In

Zenith Launcher supports multiple account types:

Microsoft Account

Full access to all Minecraft features. Requires valid Minecraft: Java Edition purchase.

Demo Mode

Try Minecraft for free with limited features and time restrictions.

Internal Account

For testing purposes. We recommend purchasing the game to support Mojang.

🎮 Custom Controls

Zenith Launcher features a fully customizable control system:

  • On-screen buttons - Resize, reposition, and customize every button
  • Joystick - Adjustable size and position
  • Gesture controls - Swipe, tap, and multi-touch support
  • Game controllers - Full MFi and Xbox/PS controller support
  • Keyboard & Mouse - Full support with customizable keybindings
