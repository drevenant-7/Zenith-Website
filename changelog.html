Changelog

Latest updates and improvements for Zenith Launcher iOS

1.3.4 "MobileGlues"

March 5, 2025

This update brings the latest MobileGlues renderer with significant improvements to performance and stability.

Improvements

  • Updated MobileGlues to v1.3.4 (iOS-specific build)
  • Significantly reduced library size (~2MB smaller IPA)
  • Minor performance optimizations
  • Compute mode for MultiDraw emulation is now stable

Bug Fixes

  • Fixed several issues that could cause crashes
  • Fixed rendering anomalies
  • Fixed known issues in Compute mode for MultiDraw emulation

1.3.3 "Stability"

February 20, 2025

Various bug fixes and improvements for a smoother experience.

Bug Fixes

  • Improved input handling
  • Control layout corrections
  • General stability improvements

1.3.2 "Renderer"

OpenGL bridge improvements and better compatibility with various Minecraft versions.

1.3.1 "Polish"

1.3.0 "Zenith"

January 6, 2025

The first release of Zenith Launcher, a personal fork of PojavLauncher iOS with custom enhancements.

Features

  • Custom dark theme with red accents
  • Internal account support
  • All PojavLauncher iOS features included
